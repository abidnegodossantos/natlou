# 🚀 Guia de Deployment em Produção - Natlou Site

## Opção 1: Vercel (RECOMENDADO - 5 minutos)

### Passo 1: Criar Conta Vercel
1. Aceder a https://vercel.com
2. Clique em "Sign Up" > "Continue with GitHub"
3. Autorize Vercel no seu GitHub

### Passo 2: Importar Projeto
1. Clique em "New Project"
2. Procure por "Cas-Maravilha/Site-Natlou"
3. Clique em "Import"

### Passo 3: Configurações
- **Framework:** Next.js (automático)
- **Clique em "Deploy"**

### Passo 4: Domínio Personalizado
1. Settings > Domains
2. Add Domain (natlou.co.ao)
3. Configure DNS records

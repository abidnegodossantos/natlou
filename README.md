## 🌟 Natlou - Site de Climatização e Vendas

Site moderno e responsivo para a **NATLOU**, empresa angolana especializada em venda, instalação e manutenção de ar condicionado.

### ✨ Funcionalidades
- **Home**: Hero com call-to-action estratégico
- **Sobre**: História, missão, visão e valores
- **Serviços**: Diagnóstico, instalação, manutenção
- **Loja**: E-commerce com produtos
- **Contacto**: Formulário + WhatsApp integrado
- **Calculadora de BTU**: Ajuda cliente a escolher AC ideal
- **PWA**: App instalável no telemóvel

### 🚀 Tecnologias
- **Framework**: Next.js 16 (React 19)
- **Linguagem**: TypeScript
- **Hospedagem**: Vercel (recomendado)
- **Build**: Static Site Generation (SSG)

### 📋 Começar

```bash
# Clonar repositório
git clone https://github.com/Cas-Maravilha/Site-Natlou.git
cd Site-Natlou

# Instalar dependências
npm install

# Desenvolvimento
npm run dev

# Build produção
npm run build
npm start
```

**URL Local:** http://localhost:3000
